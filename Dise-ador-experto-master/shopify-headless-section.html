Cómo Configurar Shopify Headless

1. Obtener Credenciales

  1. 1. Accede a tu tienda Shopify en admin.shopify.com
  2. 2. Ve a Configuración → Apps y canales de ventas
  3. 3. Crea una aplicación personalizada y genera el token de acceso privado
  4. 4. Copia el token y guárdalo en un lugar seguro

2. Configurar Variables de Entorno

Crea un archivo .env.local en tu proyecto:

VITE_SHOPIFY_SHOP_DOMAIN=feispla.myshopify.com
VITE_SHOPIFY_ACCESS_TOKEN=tu_token_aqui
VITE_SHOPIFY_API_VERSION=2024-04

⚠️ Nunca expongas el token en el código fuente. Usa variables de entorno.
